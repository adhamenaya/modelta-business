# Modelta Business Repository

## Overview

This repository maintains all **business planning, strategy, documentation, and execution files** for establishing and growing a business around the **Modelta transfer learning framework** concept.

**Note**: This is a separate repository from the technical [modelta](https://github.com/adhamenaya/modelta) framework. This repo focuses on business, marketing, operations, and strategy.

## ğŸ“ File Naming Conventions

- **Kebab-case** for file names: `market-analysis.md`, `go-to-market-strategy.md`
- **Snake_case** for directories: `customer-personas/`, `partnership-tiers/`
- **UPPERCASE** for main files: `README.md`, `INDEX.md`
- **Date prefix** for time-sensitive files: `2024-q1-roadmap.md`, `2024-annual-report.md`

3. Use provided templates from `templates/`

- Use branches for major initiatives
- Keep up-to-date quarterly reviews
- Maintain changelog of business milestones
